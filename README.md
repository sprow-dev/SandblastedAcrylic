# Sandblasted Acrylic Effect
Description: A futuristic effect designed carefully for transparent contrast.
Using a combination of a very subtle WinForm opacity effect combined with a DWM effect that has been natively supported for nearly a decade, this effect creates a defined, sharp glow to anything underneath.

## How to add to my own code?
Simple: [Go to this Gist that I created.](https://gist.github.com/sprow-dev/6befa13ca115f7968e2f5cf54b9bc6cf)

## How does it work?
It hooks into the DWM API to apply the acrylic effect at 62% intensity (dark).
Then, to add definition, it uses a slight amount of transparency in the form itself. (98% opacity to be exact)
